# mobile-app

Це простий сайт для ознайомлення з мобільним застосунком.

## Технології
- JavaScript (ES6+)
- HTML
- CSS


## Запуск проекту
1. Клонуй репозиторій:
   ```bash
   git clone https://github.com/Hrechaniuk1/mobile-app
2. Перейди на сторінку проекту
   ```bash
   cd mobile-app
4. Встанови залежості
   ```bash
   npm install
6. Запусти сервер
   ```bash
   npm run dev
